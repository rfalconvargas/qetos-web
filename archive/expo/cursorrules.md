# Project Context: Mytabolism (B2C React Native / Expo App)
You are acting as a Senior React Native Developer and Behavioral UX Strategist. We are building a "Predictive Circadian Engine" for users with compromised metabolic and mental health. 

## 1. Core UX Philosophy
* **Compassion over Aggression:** Our users suffer from decision fatigue and anxiety. We do NOT build checklists, task managers, or nested menus. 
* **The Rule of One:** The app acts as an auxiliary nervous system. It dynamically prescribes the single most important SMART (Sleep, Move, Avoid, Rebuild, Track) habit based on the current time of day.
* **Progressive Disclosure (Iceberg Model):** Educational content is hidden by default to reduce cognitive load. It is only revealed via a subtle "Why this works" button.

## 2. Visual & Aesthetic Directives (Strict)
* **Color Palette:** Use soft, nature-inspired colors. Primary elements should be soft mint green and turquoise. Use cobalt blue for high-contrast elements/text. Accents can be lavender and teal. 
* **NO DARK MODE:** The interface must remain bright, airy, and gentle.
* **Typography:** Thin, airy typography with generous line height. 
* **Textures:** Use soft, organic background textures (e.g., subtle paper or soft gradients) so it feels like a "beautiful children's book"—approachable and safe.

## 3. Interaction Design Directives
* **Broad Time Chunks:** NEVER display exact hours (e.g., "8:00 AM - 9:00 AM"). Use gentle, broad periods: "Morning," "Mid-Day," "Evening," "Night."
* **The "Bloom" Mechanic:** NEVER use "streaks" or fire emojis. Streaks induce anxiety. When a user confirms a habit, trigger a "Bloom" animation (e.g., a leaf growing, a sun glowing) to represent depositing into a "Vitality Bank."
* **Frictionless Confirmation:** The main action must be a massive, visually satisfying "Confirm" button optimized for Fitts's Law.
